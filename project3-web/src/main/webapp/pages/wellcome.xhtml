The Imixs Workflow - JEE Example Applicaton

Welcome to the Imixs Workflow Project

The Imixs Workflow Project provides you with a java based workflow framework to be used to build Workflow Management Systems (WFMS) in a fast and easy way. The Project provides different components and subprojects which should help to simplify the development of workflow functionality in any kind of java or jee application.
